3. Name two uses of a div tag?
Answer: a. The div tag creates a division or section in the HTML document in which we are working.
b. We can also design a big box and include small boxes inside it using div tag.

4. What is the difference between relative positioning and absolute positioning in HTML?
Answer: Relative: The element is positioned relative to its normal position.
Absolute: The element is positioned to its first positional parent.

5. What is the use of opacity in CSS?
Answer: Opacity in CSS can be used to set the transparency of an element. It ranges from 0 to 1.
